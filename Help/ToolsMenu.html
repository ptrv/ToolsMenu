ToolsMenu Adds some common used fucntions to the OSX menu


Inherits from: Object


Needs:  wslib quark, TabbedView quark. Only works on OSX.


Creation / Class Methods


*add (foldersToScan, foldersToShow)

adds the menu to the menu bar

foldersToScan - an array of paths (can be files or folders). Folders are recursivly searched and subfolderes added to menu.

foldersToShow - an array of paths (should be folders) which will be shown as menu items. 

// add this to your startup.rtf

ToolsMenu.add(

[ 

Platform.userAppSupportDir +/+ "startup.rtf",

"/Users/miguelnegrao/Library/Application Support/SuperCollider/Extensions/"

],

[

"/Users/peter/scwork/"

];

); 

Note: To run a  file on cmd^r change the path to the file you want to run in the sourcefile of this class.
